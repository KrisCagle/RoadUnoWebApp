# Venue Outreach

Encodes the outreach voice and structure so booking emails stay consistent and don't read like a mail-merge.

## Structure (in this order)

1. **Subject line** — specific, no hype.
2. **Opening line** — reference something specific about the venue. Never open with filler like "I hope this finds you well."
3. **The artist, in 1-2 sentences** — genre, a concrete draw-size proof point, and why this venue fits.
4. **The ask** — specific date(s), openness to alternates, and guarantee vs. door split if known.
5. **Close** — one clear next step. Never end on something vague.

## Voice rules

- Confident, brief, professional-friendly. Not desperate, not corporate.
- No exclamation points unless genuinely earned.
- No padding adjectives ("amazing," "incredible").
- Under ~150 words.

## Required input data

Artist name/genre, requested date(s)/city, draw-size proof point, venue name plus one specific detail, and budget context if relevant.

If a required data point is missing, ask for it rather than inventing a placeholder claim.
